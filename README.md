# Colorado-Car-Crash-Analysis
This project analyzes over 90,000 motor vehicle crashes across Colorado using data science and statistical techniques. By combining crash records, public school location data, and custom-collected historical weather data, I investigated how crash frequency varies with time, location, and weather conditions. The analysis identified strong relationships between crash frequency and traffic density, revealed geographic crash hotspots, and evaluated the predictive power of weather variables using machine learning models. The project involved data cleaning, geospatial analysis, statistical testing, visualization, logistic regression, and decision tree modeling in Python.

## Key Skills Demonstrated
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Hypothesis Testing
- Geospatial Data Analysis
- Logistic Regression
- Decision Trees
- Data Visualization
- Python (Pandas, NumPy, Matplotlib, Scikit-learn)
